Introduction

Welcome to the Biomass Mapping tool. This tool is designed to provide raw biomass potentials for various geographies across the world. Check out the Biomass Potential Calculator which will allow you to select the geography of your choice along with several other factors such as RCP, Climate Model etc. and allow you to obtain total biomass potential from that area. You can aslo get the predicted biomass potential from future years, and can individually get the potential from cropland or marginal land. To learn more about the working methodology, please refer to the documentation. The Data and Sources section contains all of the data that has been used to make this tool along with the sources. We will soon be adding more functionality to the tool.
